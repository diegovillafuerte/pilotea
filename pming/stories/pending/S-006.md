# Cross-Platform Comparison & Recommendations

Side-by-side comparison of metrics across platforms, the rule-based recommendation engine, and engagement features like streak tracking.

## Tasks
- B-014: Compare page
- B-015: Recommendation engine and streak tracking

## Acceptance criteria
- Compare page shows side-by-side bars per metric with winner indicator
- Only shows metrics available on both platforms being compared
- Recommendation engine generates 2-3 actionable tips per upload
- Cross-platform recommendations (e.g., "DiDi pays X% more per hour")
- Streak counter tracks consecutive weeks with uploads
- Streak badge displayed on dashboard (lightning bolt / fire emoji thresholds)
