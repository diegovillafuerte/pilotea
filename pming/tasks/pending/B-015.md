# Recommendation engine and streak tracking

Rule-based recommendation engine that generates actionable insights, plus streak counter for engagement.

## What to do

### Recommendations
1. `src/lib/recommendations/engine.ts` — generate recommendations from metrics, percentiles, and cross-platform data
2. Rules per tech design Appendix B:
   - Metric in top 25% → positive message
   - Commission above p75 → warning
   - Platform A >15% better than Platform B on a metric → actionable cross-platform tip
   - Metric below p25 → improvement suggestion
   - Streak milestones → encouragement
   - Low data_completeness → suggest better upload type
3. Max 3 recommendations, sorted by priority
4. All copy in Spanish

### Streak tracking
5. Update streak logic: on each successful upload, check if driver uploaded in the previous week
   - Yes → increment streak_weeks
   - No → reset to 1
6. Streak badge on dashboard: lightning bolt at 4+ weeks, fire at 8+ weeks
7. `src/components/dashboard/streak-badge.tsx`

### Data completeness
8. Show data_completeness indicator on dashboard when < 100%: "Datos {X}% completos — sube el PDF para mejores resultados"

## Acceptance criteria
- Recommendations generated correctly for all rule conditions
- Cross-platform recommendations work when driver has multiple platforms
- Streak increments/resets correctly based on weekly upload cadence
- Badge displays at correct thresholds
- Data completeness shown with actionable suggestion to improve
