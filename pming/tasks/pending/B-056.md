# Referral & partners program

Two proven loops: driver-to-driver referrals (give/get premium days) and the Ruta Rentable Partners model (driver-influencers paid to promote).

## What to do

1. Referral codes: each account gets a code; redemption grants premium days both ways (e.g., 14/14); abuse limits
2. Redemption at onboarding ("¿te recomendó otro conductor?") + deep link support
3. Backend: referral graph, grant ledger, basic fraud checks (device/account heuristics)
4. Partners v1 (lightweight): per-partner codes with attribution counts exportable — enough to run manual payouts to driver-creators; in-app "hazte partner" link to a form
5. Dashboard-ready metrics: K-factor inputs (invites sent, redemptions, activation)

## Acceptance criteria
- Referral lifecycle works end-to-end incl. premium grant
- Partner attribution counts accurate and exportable
- Abuse guardrails tested (self-referral, repeat devices)
