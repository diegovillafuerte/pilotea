# "Tu Mes" shareable card

Generate a branded monthly summary image that drivers can share on WhatsApp for viral growth.

## What to do

1. `src/app/api/share/card/route.ts` — GET: generate image using Satori + resvg-js. Query: ?month=2026-03
2. Card design (Satori JSX → SVG → PNG):
   - Pilotea branding (logo, name)
   - Monthly earnings total
   - Percentile ranking ("Le ganas al X% de conductores")
   - Best day (name + earnings)
   - Best platform (if multi-platform)
   - Streak badge
   - Compact, visually appealing, WhatsApp-optimized dimensions
3. `src/app/(app)/dashboard/page.tsx` — add "Tu Mes" section at bottom of dashboard (visible after month end)
4. Share button: "Compartir en WhatsApp" — uses Web Share API or WhatsApp deep link with image

## Acceptance criteria
- Image generates with correct monthly stats
- Pilotea branding visible on every shared card (free marketing)
- WhatsApp share works on Android Chrome
- Card looks good at WhatsApp image preview dimensions
- Handles missing data gracefully (don't show empty metrics)

## Dependencies
- B-010 (dashboard data), B-009 (percentile data)
