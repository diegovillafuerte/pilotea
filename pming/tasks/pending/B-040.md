# Stats screens & cost profile

The Inicio tab: today/this-week dashboards from captured data, plus the cost-profile editor that powers net math app-wide (GigU's Net Profit Calculator pattern).

## What to do

1. Inicio dashboard: header (net this week, streak badge), metric cards ($/viaje, $/km, $/hora, viajes/hora, tasa de aceptación) — port web dashboard UX to Compose
2. Day detail: per-shift timeline, offers seen vs taken, best hours/zones (zone = coarse area only)
3. Cost profile editor: rendimiento (km/L) or EV, gas price (city default, editable), maintenance/insurance/rent per period; live preview of net effect
4. Empty states for new drivers ("Activa el lector y maneja — tus números aparecen solos")
5. data-completeness indicator when hours/km partially inferred

## Acceptance criteria
- Dashboard renders fixture-generated data correctly at 360–414 px widths
- Cost profile changes recompute historical net views
- All copy Spanish, MXN formatted
