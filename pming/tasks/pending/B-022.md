# PWA setup

Finalize Progressive Web App configuration for installability and basic offline handling.

## What to do

1. `src/app/manifest.ts` — web manifest with app name, icons, theme color, display: standalone, orientation: portrait
2. Create PWA icons: 192x192 and 512x512 PNG in `/public/icons/`
3. Configure `@ducanh2912/next-pwa` in next.config.ts:
   - Service worker generation
   - Cache strategy: network-first for API, cache-first for static assets
   - Offline fallback page
4. `src/app/offline/page.tsx` — simple "Sin conexion" page shown when offline
5. Install prompt: show "Agregar a pantalla de inicio" banner after second visit (use beforeinstallprompt event)
6. Meta tags: theme-color, apple-mobile-web-app-capable, viewport

## Acceptance criteria
- App installable from Android Chrome "Add to Home Screen"
- Installed app opens in standalone mode (no browser chrome)
- Offline page shows when no connectivity
- Static assets cached for fast repeat loads
- Works on limited-data Android devices

## Dependencies
- B-001 (Next.js project), B-010 (dashboard exists to install from)
