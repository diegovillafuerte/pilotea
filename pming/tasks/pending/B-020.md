# Weekly WhatsApp reminder

Automated Monday morning WhatsApp message to active drivers prompting them to upload their weekly data.

## What to do

1. `src/app/api/cron/weekly-reminder/route.ts` — POST (protected by CRON_SECRET):
   - Query active drivers (uploaded in last 4 weeks)
   - Send WhatsApp message via Twilio to each
   - Message: "Como te fue esta semana? Sube tus datos en Kompara: {link}"
   - Log sent count
2. Pre-approve WhatsApp message template in Twilio for the reminder
3. Configure Render Cron Job: Monday 14:00 UTC (8am CDMX)
4. Respect opt-out: add notification_enabled field to drivers table if needed

## Acceptance criteria
- Cron fires on Monday mornings
- Only messages drivers active in last 4 weeks (not churned users)
- WhatsApp message delivered with app link
- Protected by CRON_SECRET
- Handles Twilio rate limits / errors gracefully (don't crash on one failure)

## Dependencies
- B-004 (Twilio WhatsApp client), B-003 (Render cron setup)
