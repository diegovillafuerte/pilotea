# Web read-only & user comms

Gracefully sunset the Next.js web MVP once the Android app is live and covers its value.

## What to do

1. Web banner + interstitial: "Kompara ahora vive en Android" with Play link; upload flow marked legacy
2. WhatsApp message to existing registered drivers (via the established Twilio sender) announcing the app + migration path
3. Existing weekly_data: exportable/migratable to Android accounts on login with the same phone number
4. Read-only mode date set; new signups disabled on web
5. Keep health endpoint + redirects alive until traffic dies

## Acceptance criteria
- Existing users notified once (no spam), with working install path
- Same-phone login on Android surfaces historical web data
- No new web signups after cutoff

## Dependencies
- Android publicly launched (post E-009 launch gate)
