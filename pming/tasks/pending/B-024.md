# Scaffold Android app & CI

Create the native Android project at `android/` in this repo: Kotlin, Jetpack Compose, Gradle version catalogs, and a GitHub Actions workflow that builds on every push.

## What to do

1. `android/` — new Gradle project: package `mx.kompara.app`, minSdk 26, targetSdk latest stable, Kotlin + Compose BOM, Material 3
2. Gradle version catalog (`gradle/libs.versions.toml`) for all deps
3. Base app: single `MainActivity`, Compose theme placeholder, app icon placeholder
4. `.github/workflows/android.yml` — assembleDebug + unit tests on push/PR; cache Gradle
5. Update root `.gitignore` for Android (build/, .gradle/, local.properties, keystores)
6. `android/README.md` — build/run instructions

## Acceptance criteria
- `./gradlew assembleDebug test` passes locally and in CI
- Project opens clean in Android Studio
- No secrets committed (local.properties ignored)

## Dependencies
- None (first Android task)
