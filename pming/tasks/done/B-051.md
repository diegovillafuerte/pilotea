# IMSS threshold tracker

Unique-to-Mexico feature from the 2025 platform-work reform: social-security coverage requires ≥ 1 monthly minimum wage (~MXN $8,364, indexed) of net monthly income per platform. Kompara tracks it automatically from captured data.

## What to do

1. Fiscal tab section: per-platform monthly net progress bar vs the threshold, days-remaining pacing ("te faltan $2,140 y quedan 9 días")
2. Threshold value remote-configured (minimum wage changes yearly; reform rules may evolve)
3. Coverage status explainer: what IMSS coverage includes for platform workers, what happens above/below threshold, pilot-program context (plain Spanish, lawyer-reviewed via B-038 cadence)
4. Month-end summary notification (covered / not covered this month, per platform)
5. Edge cases: multi-platform totals vs per-platform rule (per reform: per platform), partial months

## Acceptance criteria
- Live progress accurate from captured + imported data
- Threshold updates via remote config without release
- Explainer content reviewed against current IMSS/SAT guidance
