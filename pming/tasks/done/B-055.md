# Shareable earnings card

Port the "Tu Mes" concept, WhatsApp-native: a branded weekly/monthly card drivers actually want to share (the acquisition loop driver communities run on).

## What to do

1. "Tu Semana" / "Tu Mes" card: net earnings, trips, best metric + percentile flex ("top 22% en CDMX"), streak, subtle Kompara branding + download CTA
2. Render as image on-device (Compose → bitmap), one-tap share sheet (WhatsApp first)
3. Privacy controls: hide amounts toggle (share percentiles/streak only)
4. Entry points: week close notification, Inicio header, month summary
5. Track share events (anonymized) for loop measurement

## Acceptance criteria
- Card renders crisp at WhatsApp preview sizes
- Hide-amounts mode fully redacts earnings
- Share → install attribution path documented (Play referrer)
