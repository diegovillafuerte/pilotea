# Design system & navigation shell

Spanish-first Compose design system carrying over the web MVP's visual identity, plus the navigable app shell.

## What to do

1. `:ui` theme — colors, typography, spacing; dark mode support (drivers work nights)
2. Core components: MetricCard, VerdictBadge (traffic light), ProgressBar, EmptyState, PrimaryButton
3. Navigation: bottom nav shell — Inicio (stats) | Comparar | Lector (prominent center, like web's "Subir") | Fiscal | Ajustes
4. MXN currency + es-MX number formatting utilities
5. All copy in Spanish via string resources from day one

## Acceptance criteria

- Components previewable via Compose previews
- Verdict colors meet contrast requirements in sunlight-readable palette
