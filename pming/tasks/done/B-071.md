# In-app help center (Ayuda)

P0 gap from the 2026-06-12 benchmark: StopClub/GigU and Gridwise both have a help center; reviews
cite it as a professionalism signal. Kompara had none.

## What was done
- Ajustes → "Ayuda" (`HelpScreen`, route `help`): seven expandable es-MX FAQ entries — activar el
  lector (incl. the DiDi screen-capture step), qué significa el semáforo (two-tier floors), el
  teléfono apaga el lector (OEM battery), datos/privacidad, la cuenta, precio, y troubleshooting.
- Static/offline content (a driver mid-shift has no patience for a webview).

## Follow-ups
- A public web version (kompara.mx/ayuda) when the marketing site exists.
- A real support channel (the FAQ deliberately names none yet — nothing to point at).
