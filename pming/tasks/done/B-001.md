# Initialize Next.js project

Set up the Next.js 15 project with the full tech stack and folder structure defined in `docs/technical-design.md` section 4.

## What to do

1. Initialize Next.js 15 with App Router, TypeScript (strict), Tailwind CSS 4
2. Install core dependencies: drizzle-orm, postgres, @aws-sdk/client-s3, @anthropic-ai/sdk, twilio, jose, zod, swr
3. Install dev dependencies: drizzle-kit, vitest, eslint, prettier
4. Create the folder structure from the tech design (src/app, src/lib, src/components, src/hooks, migrations, seed, tests)
5. Configure ESLint, Prettier, tsconfig (strict mode)
6. Create `.env.example` with all required env vars
7. Set up `pnpm` scripts: dev, build, start, lint, test, db:generate, db:migrate, db:seed

## Acceptance criteria
- `pnpm dev` starts the app locally
- `pnpm build` compiles without errors
- `pnpm lint` passes clean
- Folder structure matches tech design section 4
- .env.example documents all required environment variables
