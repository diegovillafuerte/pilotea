# Comparar UI — branded hero, benchmark table, week dropdown, 4 states

Rebuild `CompararScreen` to the locked design. See `docs/comparar-redesign-spec.md` §1, §6, §7, §8, §9, §11 and the v5 mockups in `.session-captures/comparar/`.

## What to do
1. **Brand bar** — shared `KomparaWordmark()` composable (extract a `:ui` version from the private `BrandStrip` in `overlay/VerdictChipUi.kt`; uses `R.drawable.ic_kompara_logomark` + a new `brand_name` string) + a `ComparBrandBar(context = "Comparar")`.
2. **Shareable hero (FREE)** — `ShareableHeroCard`: standing sentence ("Le ganas al X% …") + `PercentileBar` + `PercentileBadge` + share icon → `navController.navigate(SHARE_CARD_ROUTE)`; degrade to the "pending" line when standing is null.
3. **Benchmark table (PREMIUM)** — `BenchmarkTable`: 6 rows (Ganancia neta, IPH, IPK, IPT, Viajes/hora, Take rate), grouped "Promedio ciudad" header over Uber/DiDi, **Tú** column emerald-tinted, percentile mini-bar + "Top X%", `Formatters.DASH` for N/A with a tiny reason, Take rate "↓ menos es mejor". (Formatting: `formatMxn`/`formatPerHour`/`formatPerKm`/`formatPerHourCount`/`formatPercent` — no `formatPerTrip`/`pct`.)
4. **Week dropdown** — replace `WeekPickerRow` FilterChips with a `ModalBottomSheet` of weeks (newest first, grouped by `formatMonthLabel`) + optional `‹ ›`; keep visible in empty state.
5. **Gating** — wrap table + opportunities (B-088) in one `PaywallGate(surface = GateSurface.COMPARE, valueHint = gate_hint_compare, ctaText = paywall_cta)`; hero stays outside the gate. Document the capability divergence (Inicio uses BENCHMARKS for percentiles; Comparar rides COMPARE).
6. **States** — empty (`EmptyState` + onOpenReader) / single-app (real percentile + both averages + "prueba la otra app") / locked (tease) / full. **Remove** `ExampleTeaser`/`exampleResult`, `CompareRowBars`/`PairedBar`/`MetricBreakdown`, `PlatformPairChooser`.
7. **Strings** — add the new `comparar_*` keys (spec §11); retire the obsolete ones; promote `titleMedium`/`labelLarge`/`bodySmall` into `KomparaTypography` (Inter).

## Acceptance criteria
- 4 states render correctly; hero free; table/opps gated with a blurred tease (never blank).
- N/A honest; week dropdown handles many weeks; Mexican Spanish; correct at 360–414 px.
- Compose previews for each state; no `FLAG_SECURE`.

## Dependencies
- B-086 (the comparison model + VM). B-088 (opportunities) renders inside the same gate. Ref spec §7–§11.
