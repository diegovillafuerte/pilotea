# MVP — Auth, Upload & Dashboard

The foundational epic. By the end, a driver can authenticate via WhatsApp magic link, complete onboarding, upload an Uber PDF, and see their dashboard with metrics and percentile rankings.

This covers Phase 1 of the technical design: project scaffolding, database, auth, the first parser (Uber PDF — richest data source), and the core dashboard experience.

## Stories
- S-001: Project Setup & Infrastructure
- S-002: Auth & Onboarding
- S-003: Upload & Parsing Pipeline
- S-004: Dashboard & Percentiles

## Success criteria
- Driver can log in via WhatsApp magic link
- Driver completes onboarding (name, city, platforms)
- Driver uploads an Uber PDF and sees extracted metrics
- Dashboard shows percentile ranking against synthetic population data
- App is deployed and accessible on Render
