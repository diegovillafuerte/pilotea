The "20 people" standing visual — filled glyphs = drivers you beat, dim glyphs = ahead of you.

```jsx
<PercentileBar displayPercentile={78} />   {/* ~16 of 20 filled */}
```

Higher = more filled, for every metric (pass the already-inverted display percentile). Carries one spoken description for the whole row. Glanceable, no per-icon noise.
