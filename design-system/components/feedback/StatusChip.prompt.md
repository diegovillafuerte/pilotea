```jsx
<StatusChip tone="success">Cubierto este mes</StatusChip>
<StatusChip tone="warning">En camino</StatusChip>
<StatusChip tone="danger">No cubierto</StatusChip>
<StatusChip tone="neutral">Importado</StatusChip>
```

Tonal tinted background + matching text. Not a verdict — use VerdictBadge for verde/amarillo/rojo verdicts.
