Shown when a screen has no data yet. Large brand icon → title → supportive body → optional primary CTA.

```jsx
<EmptyState
  icon={<HomeIcon />}
  title="Aún no hay números"
  body="Activa el lector y maneja — tus números aparecen solos."
  ctaText="Probar el lector"
  onCta={openReader}
/>
```

Copy is plain es-MX, `tú`. Centre-aligned; the CTA is full-width up to 320px.
